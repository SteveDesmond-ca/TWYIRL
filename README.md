## Contribute Lyrics

If you'd like to contribute lyrics to this bot's repertoire, please DM the bot or create an issue here.

## Requirements

- .NET SDK 6
- Twitter API keys set to the following environment variable names:
    - `ConsumerKey`
    - `ConsumerSecret`
    - `AccessKey`
    - `AccessSecret`

## Build from source

```
dotnet restore
dotnet build
dotnet test
```
